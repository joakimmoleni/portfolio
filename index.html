About

I’ve been hooked on tech since I was a kid — mixing parts that didn’t belong together, just to see what would happen.
Back then it was chaos; now it’s craft.

Today I work as a solution engineer and developer at one of Sweden’s largest banks, where I get to blend cutting-edge tools with rock-solid mainframe systems.

  • Web

    I work fluently with the modern web stack — JavaScript, TypeScript, React, Vue, Angular, Svelte, Next.js, and the tooling behind it. But when possible, I prefer to keep things lean: vanilla code, built from the ground up. Less magic, more control.

  • Mobile

    I can ship native apps and cross-platform products that work seamlessly on web and mobile. But my focus is always on keeping it lightweight — writing what’s needed, avoiding bloated stacks, and owning the code.

  • Infrastructure

    I’ve worked across the stack — Node.js, Python, Docker, Django, even COBOL and Assembler. But the tech is secondary. What I really enjoy is building the core myself, shaping infrastructure that everything else can stand on.

  • Databases

    Databases aren’t optional — they’re the heartbeat of any system. I’ve handled systems with millions of daily transactions, but I still approach them the same way: keep it lean, know exactly how the data moves, and never hide behind abstractions.
